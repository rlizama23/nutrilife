# NutriLife 🌿

App de seguimiento nutricional personalizado.

## Cómo subir a Vercel

### Opción A — Sin código (recomendada)
1. Ve a [vercel.com](https://vercel.com) y crea una cuenta gratis
2. Sube esta carpeta a GitHub (github.com → New repository → Upload files)
3. En Vercel: "Add New Project" → importa tu repositorio
4. Vercel detecta React automáticamente → clic en "Deploy"
5. En ~2 minutos tienes tu URL: `nutrilife.vercel.app`

### Opción B — Con terminal
```bash
npm install -g vercel
cd nutrilife
vercel
```
